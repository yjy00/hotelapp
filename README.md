■アプリテーマ
前職のホテルの料飲部において、キッチンとホールのコミュニケーションの課題を解決するアプリ


■概要
◇基本機能
最初にキッチンかホールかを選んでもらい、それぞれの管理画面に飛ぶ
キッチン：仕込みをしたら登録する
ホール：キッチンが仕込みをしたものを冷蔵庫から取り出す際に、いくつ取り出すか登録する。
また、新しい飲み物を開ける際に登録する。→発注リストに飛ぶ。

◇その他機能
使ったものを発注リストに飛ばす
キッチンとホール間でのチャット機能
その日の予約等確認事項を管理する機能
常連のゲストを管理する機能　など

こういった機能がコミュニケーションの解決につながると考えています。
